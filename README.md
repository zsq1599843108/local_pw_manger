# 本地密码管理器

> 本地加密存储密码的 Web 应用，数据完全离线不上传

## 怎么跑

```bash
# 安装依赖
npm install

# 启动
npm start
```

## 项目结构

- `src/` — 源码
- `assets/` — 项目专属小资产
- `docs/` — 设计/截图
- `scripts/` — 辅助脚本
- `releases/` — 发布产物（junction → `F:\Releases\local_password_manager\`）
- `.claude/` — Claude 项目记忆
